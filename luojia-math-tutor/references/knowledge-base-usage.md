# 珞珈数智助教本地知识库使用指南

为了提供最权威、准确的解答，本技能内置了丰富的本地知识库（“珞珈数学库”）。作为助教，你在“概念讲解”和“思路规划”时，必须优先检索并参考这些本地资料。

## 1. 结构化概念题库 (JSON)
路径：`references/output/`

该目录下包含分层级的核心定义、定理和推导过程的三元组数据。
*   **`concept_gs.json`**：高等数学（微积分）相关概念。
*   **`la.json`**：线性代数相关概念。
*   **`proba.json` / `Proba_example.json`**：概率论与数理统计相关概念。

### 数据结构示例
每条记录包含如下核心字段：
*   `concept_zh`：概念/定理名称。
*   `prerequisite`：前置知识。
*   `description` / `core_content_latex`：标准学术定义（LaTeX 公式）。
*   `intuitive_explanation`：直观解释（用于给学生做通俗类比的基础）。

### 使用方法
1.  当学生询问“什么是 XXX”或你在解题需要引用某定理时，使用文件内容搜索工具（如 `grep_search`）在此目录下检索关键词（如 `grep_search("二阶行列式", "references/output/la.json")`）。
2.  提取出 `description` 和 `intuitive_explanation`，结合《交互与启发式引导规范》输出给学生。

## 2. 权威教材库 (PDF)
路径：`references/textbook/`

该目录下存放了各大高校现行的权威数学教材（如《概率论与数理统计(茆诗松)》、《高等代数学》等）。

### 使用方法
当 JSON 库中的内容不足以应对极其冷门、深奥或需要追溯具体章节的知识点时，可以：
1.  阅读目录定位相关教材。
2.  若需深度讲解，可引导学生查阅指定教材的某一部分，或者利用代码工具解析 PDF 提取内容作为背景知识补充。
*(注意：日常答疑优先使用 JSON 结构化数据，PDF 作为兜底的权威参考备份。)*
